格心日记——2022年09月24日

2022-09-24 2 min read

现在是周六早上11点，可以说一个愉快的周末才刚开始。

从手机网络世界逃离出来，想要充实的度过周六周日这两天。

而不是沉迷在虚拟的网络世界里。

自私用智？

然后心中想了下要做的事，感觉好多，突然不知道该干啥了：

生活

[] 逃离热闹的虚拟世界，回到单调甚至有点冷清的现实世界；【不躲避外世，静中涵养】
[] 想要整理房间，搞卫生，收拾电路（插线板不良问题），拾掇天然气，调节火焰；
[] 洗脸、刷牙、洗澡、跑步锻炼，逛西溪湿地，骑行；
[] 想要做有趣的早餐/午餐，米酒，烧水泡茶；
[] 想要买菜，剪发，买眼药水，买米酒曲；

心学修道

[] 想要继续整理、学习心学，修道；- 已知之天理不肯存，已知之人欲不肯去，一味贪多慕大；
[] 想要写关于心学修道主题的文章，小说；- 只是为了让修道更有趣，更容易接受；
[] 想要制作关于修道，心学相关的网站，APP，整理修道相关的短视频等；
[] 想要练习书法，静心，发布书法视频；

技术开源

[] 想要用富文本编辑器，复活自治领APP;
[] 想要学习VUE技术，JAVA技术，不要技术设限，应在项目中成为全领域能手；
[] 想要完善已开源的组件库和文档；
[] 想要写技术博客翻译技术文章，写掘金小册子；
[] 想要制作个人或商业项目，量化交易等；
[] 想要整理ruanyifeng周刊内容，或整理掘金沸点有趣内容，生成技术自媒体——短视频，直播；

其他

[] 想要准备考试；
[] 想要学习会计；
