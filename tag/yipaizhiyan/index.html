格心日记——20220718，空与不空
2022-07-18 4 min read # 格心日志 # 一派之言

空的妙用，或能人生开挂

因为空，因为无我。所以面对的一切都是事物本身，所做的都是1+1=2的积累，遇到不知道的部分就会去搜索学习从而能补上，假以时日，万丈高楼也能平地起，外人看来就像开了挂一样，无所不知，无所不能，无往不胜。

也就是所谓的至诚如神，因为至诚，所以眼中心中全是事物本身，没有一丝不懂装懂，没有一丝急于求成，没有一丝懈怠懒惰，然后顺着规律执行，不懂就学不懂就问，然后就无所不懂，既然无所不知自然就如神了，其一个最关键的地方就在于无我。

万物一体是什么意思？
2022-07-07 2 min read # 一派之言

天地万物时空宇宙都和你是一体的。

为什么是一体的呢，因为道，因为一切都循道而行……

  • 诋毁、赞誉，有和没有都不影响你做或不做某事；

举世誉之而不加劝，举世非之而不加沮，不以物喜…

  • 吃饭和睡觉你的心该是一样的；

  • 工作和生活你的心该是一样的；

  • 任何时间任何地点你的心也该是一样的。

  • 当喜怒哀乐或是七情六欲发生时你的心也该是一样的。

格心一派之言
2022-06-06 3 min read # 一派之言

缘起

从王阳明天泉证道四句教起言，行知行合一之教；修心外无物、万物一体之道；以恢复心中良知，使其无纤毫私欲遮蔽为格心，乃至于得道。

从另方面讲就是将阳明心学褪去儒家（包含其中禅宗、道家）外衣，只宗一个天理 (道) ——致良知。其目的在于引导有心人，修身以恢复良知去除遮蔽，悟道乃至于达道，也就从物欲横流、浮躁浅薄的琐碎现实生活中超脱出来，与天地万物同生为一，成为一个得道高人。

无私我，大公我。
体悟良知，衍行大道。
存天理，去私欲。
